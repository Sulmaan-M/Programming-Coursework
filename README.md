How to compile and run:
1. Open CLion project folder "Programming Coursework"
2. Ensure the power_quality_log.csv is inside the "cmake-build-debug" file
3. return to project folder
4. open "main" in project folder
5. IDE should open
6. click green Run button at the top of the IDE
7. review results, the results will print to console and open the txt file
